# スマレジアプリ・スターターパック
スマレジのアプリマーケットにて公開できるアプリのテンプレートです。

# 構成
- MacOSでの開発を想定しています（スクリプト構成のため）
- Python3.8ベース
- メインコンポーネント：Cloud Function＋Firestore
- サブコンポーネント：Cloud Storage（実行バイナリの保存先）、Error Reporting（アラート用）
